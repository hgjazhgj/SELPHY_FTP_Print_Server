# Privacy Policy

Last Updated: June 2026

SELPHY FTP Print Server respects your privacy.

## Information Collection and Use

SELPHY FTP Print Server does not collect, store, transmit, or share any personal information.

The application operates entirely on your device and does not require an internet connection.

## Data Storage

SELPHY FTP Print Server does not collect, store, or share personal information with the developer or any third parties.

Any information exchanged with local network devices remains within the user's local network environment.

## Third-Party Services

SELPHY FTP Print Server does not use any third-party analytics services, advertising networks, tracking tools, or cloud services.

## Internet Access

SELPHY FTP Print Server communicates only with devices that the user chooses to connect to on the local network (LAN).

The application does not transmit data to the developer, external servers, analytics platforms, or advertising services.

All communication occurs directly between the user's device and the selected local network devices.

## Children's Privacy

SELPHY FTP Print Server does not knowingly collect any personal information from children or any users.

## Changes to This Privacy Policy

If this Privacy Policy is updated in the future, the revised version will be published with an updated effective date.
